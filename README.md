# Mangalam Bakeries - Premium Valentine-Themed Bakery Website

🎂 **Premium, modern, and emotionally engaging bakery website** designed to dominate Valentine Week sales and local SEO in Bhopal.

## 🌟 Features

### 💕 Valentine-Themed Design
- Romantic color scheme (red, pink, white, gold)
- Floating hearts animations
- Glowing buttons and confetti effects
- Countdown timer for Valentine's Day
- Valentine Week calendar (7 special days)

### 🎯 Core Functionality
- **Custom Cake Builder** - Design personalized cakes with photos, messages, themes
- **E-commerce System** - Cart, wishlist, checkout, multiple payment options
- **Delivery Management** - Same-day & midnight delivery with location checker
- **WhatsApp Integration** - Quick ordering with pre-filled messages
- **Product Categories** - Cakes, Pastries, Chocolates, Gift Hampers
- **Advanced Filters** - Price, flavor, egg/eggless, occasion

### 🚀 SEO Optimization
- **Hyper-Local SEO** for Bhopal (MP Nagar, Arera Colony, Kolar Road, BHEL)
- Schema.org markup (LocalBusiness + Bakery)
- FAQ Schema for rich snippets
- Dynamic sitemap generation

- Mobile-first responsive design

## 🛠️ Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **State:** Zustand
- **SEO:** next-seo
- **Icons:** Lucide React

## 📦 Installation

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```
